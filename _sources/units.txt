Data Type Units
===============

The units of data stored by Curb is **Watts**, **Milli-Volt**, and **Volt
Ampere**. The primary unit value most users will be reading from the
API will be in Watts, however.

Basic transformations on the units of the values can be specified in
the requests to the API, please refer to :doc:`transformation`.
